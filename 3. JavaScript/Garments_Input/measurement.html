操 作 說 明：

  • 使用 Tab 鍵可快速切換輸入欄
  • 輸入錯誤可使用欄位右邊「刪除」鍵刪除該筆資料
  • 重新整理網頁，或鍵盤按「F5」清空所有資料
  • 對畫面點選右鍵並點選「另存新檔」可儲存資料表
  • 建立好的資料表可直接選取並貼入Excel中做資料分析
  • 修改: 第一欄輸入"欄號"(例：姓名 => 3, 肩寬 => 5)
  • 修改: 第二欄輸入"修改值"
  • 確認欄號及修改值後按下「修改」即可修改單欄資料
  • 於基本資料欄位填入姓名或編號，按下「搜尋資料」即可定位資料欄
